Spotify + Netflix Free

Plataforma: Android

Tecnología utilizada: Java, Klotin, Android Studio

Se trata de una versión gratuita de las aplicaciones de Streaming: Netflix y Spotify, el primero es un directorio enlazado a sitios de películas con contenido gratuito de Netflix con la capacidad de reproducir, el segundo: Spotify Free es una versión de Spotify crackeada que permite acceder al contenido de la plataforma de manera gratuita. Esta última ya no tiene soporte.

The Maze

Plataforma: Windows

Tecnología Utilizada: Unity 3D, Blender, C#, JavaScript

Es un video juego de terror en 3D ambientado en el colegio Benigno Malo, fue presentado a mediados del 2018 como proyecto final de Emprendimiento y Gestión. Para realizar y animar los assets del juego se uso Blender 3D y para las texturas del juego Photoshop Express, el código en su mayoria fue escrito en C#, sin embargo y para facilitar el proceso de desarrollo se empleo JavaScript. El juego fue descontinuado debido a que las nuevas versiones de Unity 3D ya no admitian código de JavaScript y no se contaba con el presupuesto para licencias.

S.A.M

Plataforma: Linux

Tecnología Utilizada: Python

Pese a no ser un proyecto complejo ni innovador fue un proyecto interesante, pues fue mi primer proyecto relacionado con la encriptación de datos. Su funcionamiento es simple, toma un archivo y lo cifra protegiéndolo con una contraseña, el archivo sigue disponible pero no puede ser leído a menos que se se introduzca una contraseña válida para decodificarlo. Este proyecto funciona en la terminal de linux y todavía se mantiene en desarrollo.

Plataforma de Audiolibros

Tecnología Utilizada: React, Django, Django Rest
